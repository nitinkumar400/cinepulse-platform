Multi-Server Auto-Embed

Automatic video streaming with 6+ embed servers. If one server fails, the system automatically switches to the next available server. Just enter a TMDb ID and start watching!

How It Works

  • Automatic Failover: If a server fails to load within 8 seconds, the system automatically tries the next server.
  • 6+ Servers: VidSrc, 2Embed, AutoEmbed, VidLink, SuperEmbed, and MultiEmbed.
  • TV Show Support: Select season and episode numbers for series playback.
  • TMDb Integration: Uses TMDb (The Movie Database) IDs for universal content matching.

Local Dev Mode (iframe works!)

For pure local testing, run Chrome with web security disabled. This allows embeds to load in the iframe on localhost.

Option 1: Double-click (Easiest)

Run this file in your project folder:

start-dev.bat

Option 2: Manual Command

Close all Chrome, then run in PowerShell:

& "C:\Program Files\Google\Chrome\Application\chrome.exe" --disable-web-security --disable-site-isolation-trials --user-data-dir="$env:TEMP\cinestream-dev" http://localhost:5001

⚠️ Only use for local dev! Don't browse sensitive sites in this Chrome window.

Test on HTTPS (ngrok)

For production-like testing with HTTPS. Creates a public HTTPS URL that embed servers trust.

  1. Install ngrok: npm install -g ngrok
  2. Run: ngrok http 5001
  3. Copy the https:// URL (e.g., https://abc123.ngrok.io)
  4. Open that URL + /embed-demo
